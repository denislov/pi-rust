## Scope And Definition Of Done

v1.0 converges every first-party live-session operation on the typed,
admitted `CodingAgentSession::run(CodingAgentOperation)` path, exposes the
stable `pi_coding_agent::api` facade, removes the replaced broad workflow
methods, preserves durable/event/control behavior, and closes Stage 9 with
reproducible evidence. Typed `ProductEvent` payload convergence and
compatibility-subscription deletion remain Stage 10 work.

## End-To-End Flows

The integration audit verified 8/8 flows:

1. JSON prompt through canonical admission, ProductEvents, output ordering and persistent log.
2. Persistent print continuation/fork through final text/error and replay.
3. Transient print without session artifacts.
4. RPC prompt/agent/team/delegation with response ordering, controls, replay and owner restoration.
5. RPC mutation/plugin/self-healing/profile operations with typed outcomes and restoration.
6. Interactive prompt/background/navigation with event projection and failure continuity.
7. Durable `PartialCommit`/replay behavior with operation identity.
8. External API privacy and regression guard matrix.

Fresh integration evidence: 193 focused tests passed with 0 failures.

  contract is tightened in a future milestone.
