<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Public connection -> session-owned services | An immutable handle carries id/generation into snapshot, replay, ack, and mutation paths. |
| EventService -> external client | Private sequenced events are projected into public values across replay/live recovery. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-08-04-STALE | Elevation of Privilege | all connection methods | high | mitigate | Central generation guard before reads/mutations; public tests enumerate old handle snapshot/ack/draft/submitted failures. |
| T-08-04-HANDOFF | Tampering / Repudiation | reconnect | high | mitigate | Consume EventService atomic boundary, sequence-partition tests, explicit ack, and typed recovery evidence. |
| T-08-04-PRIVATE-EVENT | Information Disclosure | public replay/live wrapper | high | mitigate | Convert to CodingAgentProductEvent before API crossing; boundary tests reject ProductEvent/receiver leakage. |
| T-08-04-CURSOR-LOSS | Tampering | acknowledge | high | mitigate | Delivery cannot mutate ack; validate monotonic bounded sequence and test disconnect-before-ack replay. |
</threat_model>
