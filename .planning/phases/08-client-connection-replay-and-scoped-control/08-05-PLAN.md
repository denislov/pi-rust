<interfaces>
`CodingAgentClientConnection { coordinator: Arc<SnapshotCoordinator>, client_id, generation }` invokes named coordinator/facade methods for snapshot, reconnect, acknowledge, drafts, and control. It has no callback and no session borrow. The sole owner-requiring exception is `prepare_submission(&self, session: &mut CodingAgentSession, draft_id: CodingAgentDraftId, operation: &CodingAgentOperation) -> Result<CodingAgentSubmissionLease, CodingSessionError>`; this only installs a session-wide lease, and the caller then invokes unchanged `session.run(operation)`.
</interfaces>
