<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Public control capability -> private Prompt sender | Caller attempts to enqueue into an active operation control channel. |
| ClientService receipt cache -> retry caller | Cached authority/result must be scoped and bounded without cross-target confusion. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-08-CONTROL-AUTH | Spoofing / Elevation of Privilege | control admission | high | mitigate | Validate owner client, current generation, and exact immutable Prompt operation id before sender access; cross-client/takeover tests. |
| T-08-CONTROL-REPLAY | Tampering / Repudiation | receipt cache | high | mitigate | Scope key by client+target+control id, return original receipt, enqueue once, test response-loss retry and distinct ids. |
| T-08-05-RETARGET | Tampering | CodingAgentPromptControl | high | mitigate | Store immutable target operation id; reject mismatch/not-running, never resolve a later current Prompt. |
| T-08-05-QUEUE | Denial of Service | Prompt control transport / receipt records | high | mitigate | Replace raw unbounded admission with a bounded private channel/permit cap; return typed `QueueCapacityExceeded`, preserve drafts, retain bounded FIFO receipts, and test saturation, ordering, closed-channel rejection, and response-loss retry. |
</threat_model>
