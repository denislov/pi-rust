## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| model/tool event -> ProductEvent | Untrusted text, tool arguments, errors, and provider metadata become an owned typed payload. |
| ProductEvent -> public Serde envelope | Internal event data crosses the stable facade and may be consumed by external clients. |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|

| T-07-02 | Information Disclosure | `public_event.rs` facade projection | high | mitigate | Keep internal `CodingAgentEvent`, services, and raw storage private; public tests bind only exported typed payloads. |
| T-07-03 | Spoofing | operation/terminal metadata | medium | mitigate | Derive identity and the five terminal associations from typed variants at the event boundary; never accept adapter-supplied metadata. |
</threat_model>
