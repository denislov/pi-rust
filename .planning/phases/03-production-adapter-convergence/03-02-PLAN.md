<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| JSONL command -> admitted background operation | Untrusted command data selects profiles, teams, delegation targets, and prompt work. |
| Runtime ProductEvent -> bounded RPC queue -> wire event | Event loss/reordering or overflow handling affects client-visible protocol truth. |
| RPC control command -> live PromptControlHandle | Abort, steer, and follow-up mutate an active operation outside ordinary operation submission. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-03-05 | Tampering / Elevation of Privilege | RPC operation admission | high | mitigate | Construct only public typed operations after existing validation and preserve dynamic delegation target resolution. |
| T-03-06 | Repudiation / Denial of Service | event queue, cursors, and final drain | high | mitigate | Keep bounded queue, response/event ordering, sequence cursors, overflow recovery, and final drains unchanged and behavior-tested. |
| T-03-07 | Tampering / Denial of Service | abort/steer/follow-up routing | high | mitigate | Retain PromptControlHandle and the existing select branches; do not fold controls into operations. |
| T-03-08 | Information Disclosure | RPC error/response projection | medium | mitigate | Preserve exact wire fields and existing typed error conversion; add no new diagnostics. |
</threat_model>
