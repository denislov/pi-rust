# 0.5.3 Release Evidence

Release date: 2026-07-21. Workspace package version: `0.5.3`.

## Runtime Evidence

- One canonical interactive client connection owns typed snapshot,
  replay-before-live delivery, retained-gap recovery, and acknowledgement.
- Stdin uses a 32-chunk bounded channel; operation controls use reviewed bounded
  queues; the interactive/runtime source inventory contains zero unbounded
  channels.
- Running and idle modes share one typed select loop. Unix uses `SIGWINCH`; the
  100 ms compatibility fallback suppresses unchanged sizes.
- `Tui` balances synchronized output after normal writes, fail-on-Nth I/O,
  panic unwind, image cleanup failure, resize, overlay, and stop retry.
- The fixed `0.5.3-mixed-v1` offline soak schedule passed 20 iterations and
  1,020 focused cases with per-command deadlines.
- The real TypeScript/Wasm lease-backed Host vertical slice passed in `174.78s`
  under the existing approximately `187s` debug cold-path baseline.

## Dead-Code Inventory

The Rust-source inventory fell from 68 to 23 occurrences and contains zero
unreasoned exceptions:

- 6 default-production architecture exceptions: three retained ADR-002
  Extension runtime scopes and three item-level core handler target items;
- 3 non-default/test-only `pi-coding-agent::test_support` impl exceptions;
- 14 Cargo shared integration and compile-fixture exceptions.

Provider fields are now consumed/validated or deleted. The baseline script
fails if an unreasoned dead-code allowance is introduced.

## Protocol And API

RPC `2.1`, ProductEvent `2.2`, and UI snapshot `2.1` are unchanged. Normalized
public API artifacts are generated under
`target/release-artifacts/0.5.3/public-api/`; the checked-in manifest is
`docs/api-snapshots/0.5.3/SHA256SUMS`.

The normalized `pi-ai`, `pi-agent-core`, and `pi-coding-agent` API files are
checksum-identical to 0.5.2. `pi-tui` differs only because `ProcessTerminal`
now has positive `UnwindSafe` and `RefUnwindSafe` auto-trait implementations
after removal of its progress thread/atomic state.
