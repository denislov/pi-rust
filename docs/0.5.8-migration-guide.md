# Migration Guide: 0.5.7 to 0.5.8

0.5.8 changes only local fullscreen refresh and compact Context presentation.
No application integration, persistence migration, or protocol negotiation
change is required.

## Fullscreen Status And Operations

The running spinner and Context operation elapsed labels now refresh from an
independent 120ms main-loop tick. Capture tests should no longer assume that
the spinner frame or a running elapsed value remains stable between frames.

Compact Context Operations rows no longer contain the operation ID. Open the
selected detail surface to inspect the full operation, parent, and root IDs.
Machine-readable ProductEvent, snapshot, JSON, and RPC representations are
unchanged.

## Protocol Inventory

| Family | 0.5.7 | 0.5.8 |
| --- | --- | --- |
| RPC | `2.1` | `2.1` |
| ProductEvent | `2.2` | `2.2` |
| UI snapshot | `2.2` | `2.2` |

Public Rust APIs and all persistence/Extension schemas are unchanged.
