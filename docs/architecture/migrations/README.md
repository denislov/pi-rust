# Architecture Migration Records

Migration records preserve superseded structures, compatibility decisions,
source baselines, and completion evidence. They are not active normative
contracts.

- [`0.3.1-monolithic-architecture.md`](0.3.1-monolithic-architecture.md) preserves
  the pre-split architecture document and detailed implementation inventory.

Each completed release adds or updates a record containing the source range,
public/API protocol changes, durable-data decision, debt closure, validation, and
superseded ADR history.
